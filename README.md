# Cafe De Piero

🇹🇷 **Türkçe**
Modern, zarif ve tamamen duyarlı (responsive) bir kafe web sitesi konsepti. Bu proje, premium bir dijital deneyim sunmayı amaçlayan, hızlı, estetik ve kullanıcı dostu bir arayüz yaklaşımıyla geliştirilmiştir.

🇬🇧 **English**
A modern, elegant, and fully responsive café website concept designed to deliver a premium digital experience with clean visuals, smooth interactions, and high performance.

---

# Proje Hakkında | About The Project

## 🇹🇷 Türkçe

Cafe De Piero, yalnızca standart web teknolojileri kullanılarak geliştirilmiş front-end odaklı bir web projesidir. Projede herhangi bir framework veya harici JavaScript kütüphanesi kullanılmamıştır.

Amaç; modern, premium ve estetik bir kafe deneyimini dijital ortamda güçlü bir kullanıcı deneyimi ile sunarken, aynı zamanda yüksek performanslı ve sürdürülebilir bir yapı oluşturmaktır.

Site; responsive yapı, kullanıcı etkileşimleri, semantik HTML kullanımı, SEO dostu yapı ve modern animasyonlar içermektedir.

## 🇬🇧 English

Cafe De Piero is a front-end focused web project built entirely with standard web technologies, without using frameworks or external JavaScript libraries.

The goal of this project is to provide a modern, premium, and visually refined café experience while maintaining performance, accessibility, and maintainability.

The website includes responsive layouts, smooth interactions, semantic HTML, SEO-friendly implementation, and modern UI animations.

# Proje Yaklaşımı | Project Philosophy

🇹🇷
Bu proje hafif, hızlı ve sürdürülebilir bir front-end mimarisi hedeflenerek geliştirilmiştir.

* Framework yok
* UI kütüphanesi yok
* JavaScript bağımlılığı yok
* Tamamen statik yapı
* Yüksek performans
* Kolay bakım ve geliştirme

🇬🇧
This project follows a lightweight and maintainable front-end approach.

* No frameworks
* No UI libraries
* No JavaScript dependencies
* Fully static architecture
* High performance
* Easy maintenance and scalability
